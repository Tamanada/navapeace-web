⚠️ HOW MODULE TOGGLES WORK

Settings are saved to localStorage under the key nava_modules.
To sync across all devices and users, create an admin_settings table in Supabase:

CREATE TABLE admin_settings (key TEXT PRIMARY KEY, value TEXT DEFAULT 'true');
ALTER TABLE admin_settings ENABLE ROW LEVEL SECURITY;
CREATE POLICY "public_read" ON admin_settings FOR SELECT USING (true);

Then each app page will fetch the toggle states on load and hide/show sections accordingly.
